Hashes
Key / Value pair data type
Hash is created using curly braces and the "hash rocket": { 'key' => 'value' }
Keys and values can be any type we want
numbers, strings, arrays, other hashes
Hashes: Teacher Notes
Hashes are a datatype that allow us to group our data into names
If we want to get data out of an array, we have to know what index the data is in
array = ['data', 'data', 'data we want', 'more data']
Hashes allow us to name the data so we can access it directly
code
hash = { 'key' => 'value' }
Hash rocket
hash = { 'name1' => 'data', 'name2' => 'more data', 'name3' => 'data we want'}
hash['name3']
Used the same way as arrays using square brackets, but instead of index, we use the key
We can only get the values if we know the keys
